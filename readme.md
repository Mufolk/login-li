# Desafio de Login da Loja Integrada

## Instruções gerais

_Qualquer dúvida, favor entrar em contato_

**Instruções gerais para a execução do projeto**

- Para executar:
  npm run dev
- Para que o servidor esteja devidamente conectado, configure o arquivo config.js na raíz do server. _Num caso de produção esse arquivo jamais estaria sendo subido para o git, constando no .gitignore. Para fins de evitar problemas na execução o arquivo já está corretamente configurado com acesso ao banco de dados com usuário e senha_
